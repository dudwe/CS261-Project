Install Guide:
To run the application a LAMP(Linux,Apache,MySQL and PHP) stack is required. 
Unzip the folder and place it into the public html folder.
Create a database table called "traderbot_db" through the MySQL console
Modify the globalvars.php file to match your MySQL credentials
Run reset.php via a browser
The application has now been setup, to access the application open the UserInterface folder via the browser.
